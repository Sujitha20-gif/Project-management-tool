# Project Management Tool

This project is a mini project management web application built as part of Task 3.

## Features
- Create projects
- Role-based access (basic)
- Task structure (basic)
- Backend API using Express.js

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js

## How to Run
1. Install Node.js
2. Run `npm install`
3. Run `node server.js`
4. Open index.html in browser
